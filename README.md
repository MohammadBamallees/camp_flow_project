# 🏕️ مشروع كشتات - Camp Flow Project

### 📝 وصف المشروع
نظام متكامل لإدارة مستلزمات التخييم، يهدف إلى تسهيل عملية البيع والشراء وتنظيم المخزون.

---

### 🚀 المكونات الأساسية
* **تطبيق العميل (camp_flow):** لتصفح المنتجات، إضافتها للسلة، وإتمام الطلبات.
* **لوحة تحكم الأدمن (camp_flow_admin):** لإدارة المنتجات، متابعة الطلبات، وتحديث حالة التوصيل.

---

### 🛠️ التقنيات المستخدمة
* **Framework:** Flutter
* **Backend:** Firebase
